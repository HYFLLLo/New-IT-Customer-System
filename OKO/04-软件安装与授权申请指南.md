## 3. 常用软件申请指南

### 3.1 办公软件

#### Microsoft Office 365

**状态：** 公司已购买全员授权

**安装步骤：**

```
1. 访问：www.office.com
2. 点击"登录"，使用企业邮箱账号
3. 输入公司邮箱地址 → 下一步
4. 输入密码（与邮箱密码相同）
5. 点击"安装Office" → "Office 365应用"
6. 下载安装包后运行安装
7. 安装完成后，打开任意Office应用
8. 如提示登录，输入公司邮箱和密码
9. 激活成功！
```

**包含应用：** Word、Excel、PowerPoint、Outlook、Teams、OneDrive

**授权数量：** 5台设备/员工

#### WPS Office

**使用场景：** Mac用户、喜欢WPS界面者

**申请：** 无需申请，免费使用

**下载地址：** www.wps.cn （官网免费版）

---

### 3.2 设计与创意软件

#### Adobe Creative Cloud

**适用人群：** 设计部、市场部

**申请流程：**
1. 部门主管审批申请理由
2. IT核查公司是否还有可用授权
3. 如有授权，IT发送Adobe企业账号激活链接
4. 员工自行下载安装

**包含软件：**
- Photoshop（图片编辑）
- Illustrator（矢量图形）
- Premiere Pro（视频编辑）
- After Effects（特效）
- XD（UI/UX设计）

**注意：**
- Adobe软件较大，安装需要30-60分钟
- 建议在公司网络下安装
- 每个授权只能同时激活2台设备

## 4. 禁止安装的软件

### 4.1 严格禁止的软件

以下软件**绝对禁止**安装在公司电脑上，违者将受到纪律处分：

```
┌─────────────────────────────────────────────────────────────┐
│                     禁止安装的软件                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【网络安全类】                                             │
│  ✗ 任何翻墙/VPN软件（非公司VPN）                            │
│  ✗ Tor浏览器                                                │
│  ✗ 代理软件（如Proxifier）                                  │
│                                                              │
│  【下载类】                                                 │
│  ✗ 迅雷（严重占用公司带宽）                                 │
│  ✗ 电驴、比特彗星等P2P下载软件                               │
│  ✗ 各类下载加速器                                            │
│                                                              │
│  【娱乐类】                                                 │
│  ✗ 游戏模拟器（如夜神、雷电）                                │
│  ✗ 网络游戏（如Steam、Epic游戏平台）                        │
│  ✗ 视频播放器（非必要）                                      │
│                                                              │
│  【系统类】                                                 │
│  ✗ 磁盘分区工具（如DiskGenius）                             │
│  ✗ 系统优化/清理工具（如鲁大师、360安全卫士）                │
│  ✗ 虚拟机软件（VirtualBox、VMware）（需特别审批）            │
│                                                              │
│  【其他】                                                   │
│  ✗ 杀毒软件（非公司采购品牌）                               │
│  ✗ 解压缩软件（非授权品牌）                                  │
│  ✗ 来路不明的第三方软件                                      │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 为什么禁止安装这些软件？

| 软件类型 | 风险 |
|---------|------|
| 翻墙软件 | 信息安全风险，违法法规 |
| P2P下载 | 严重占用带宽，影响全公司网络 |
| 游戏软件 | 影响工作效率，可能携带病毒 |
| 系统优化类 | 可能破坏系统稳定性，泄露数据 |
| 非授权杀毒 | 与公司已有的杀毒软件冲突 |

## 5. 软件使用规范

### 5.1 软件使用原则

1. **工作必需才安装**：只安装工作所需的软件
2. **使用公司授权版本**：不要使用盗版/破解版
3. **及时更新**：保持软件为最新版本（安全补丁）
4. **离职前卸载**：离职时卸载所有个人软件

### 5.2 软件版本管理

```
┌─────────────────────────────────────────────────────────────┐
│                    软件版本更新流程                           │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  定期检查更新 ← 员工责任                                    │
│        ↓                                                     │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ 建议更新：                                            │    │
│  │ • Chrome/Edge/火狐浏览器                              │    │
│  │ • Office套件                                          │    │
│  │ • PDF阅读器                                          │    │
│  │ • 通讯工具（钉钉、企业微信）                          │    │
│  └─────────────────────────────────────────────────────┘    │
│        ↓                                                     │
│  不建议频繁更新（可能破坏兼容性）：                           │
│  × 打印机驱动（除非故障）                                   │
│  × Adobe全家桶                                              │
│  × CAD软件                                                  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 5.3 软件许可证管理

**个人许可证 vs 公司许可证**

| 类型 | 说明 | 归属 |
|------|------|------|
| 公司许可证 | 公司统一购买 | 公司财产，离职不带走 |
| 个人许可证 | 个人付费购买 | 个人财产，可保留 |

**注意：**
- 使用公司邮箱注册的账号，**属于公司财产**
- 离职时应将账号权限转移给接替者
- 不要用个人邮箱注册工作相关软件

## 6. 常见问题解答

### Q1: 申请的软件多久能批下来？

**回答：**

| 软件类别 | 审批时间 | 说明 |
|---------|---------|------|
| A类（自助安装） | 即刻 | 可直接安装 |
| B类（部门审批） | 1个工作日 | 部门主管审批后IT安装 |
| C类（双重审批） | 3个工作日 | IT+部门总监审批 |

**加急申请：** 如有紧急工作需要，可电话联系IT说明情况，IT会尽快处理。

---

### Q2: 公司没有购买的软件可以申请吗？

**回答：** 可以申请，但需要以下流程：

1. 员工提交申请，说明软件用途和必要性
2. IT评估软件的安全性和授权情况
3. 如为免费软件，可直接批准
4. 如为付费软件：
   - 部门预算内：部门总监审批后购买
   - 部门预算外：需另行申请专项预算

---

### Q3: Mac电脑能安装Windows软件吗？

**回答：** 部分软件支持Mac，但有些只能在Windows运行。

**解决方案：**

```
Mac用户软件建议：
┌─────────────────────────────────────────────────────────────┐
│  Windows独占软件 → 使用方案                                 │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 使用公司提供的Windows虚拟机（联系IT申请）               │
│     - 已安装常用Windows软件                                  │
│     - 需要VPN才能连接内网                                   │
│                                                              │
│  2. 使用Mac原生替代软件                                     │

│                                                              │

```

### Q4: 安装软件后电脑变慢了怎么办？

**回答：** 按照以下步骤排查：

```bash
# 步骤1：检查启动项
Ctrl+Shift+Esc → 启动
# 禁用不必要的启动项

# 步骤2：检查后台进程
# 打开任务管理器，查看CPU和内存占用
# 结束高占用进程（如果不是必要的系统进程）

# 步骤3：卸载不常用的软件
控制面板 → 程序和功能
卸载不常用的软件

# 步骤4：运行杀毒扫描
确保没有病毒/恶意软件

# 步骤5：如问题持续，联系IT重装系统
```

---

### Q5: 软件试用期到期了怎么办？

**回答：**

1. **如果是公司已授权软件**
   - 联系IT，IT会提供完整授权

2. **如果是试用版需要续期**
   - 联系软件厂商购买正式授权
   - 公司报销流程：员工先垫付 → 报销

3. **如果不需要继续使用**
   - 立即卸载，删除相关账号数据

---

### Q6: 可以安装破解版/盗版软件吗？

**回答：**

```
⚠️ 严禁使用盗版/破解软件！

违反后果：
1. 法律风险：侵犯软件著作权，公司可能被起诉
2. 安全风险：盗版软件通常携带病毒、木马
3. 职业风险：违规使用盗版属严重违纪
4. 工作受影响：盗版软件不稳定，可能丢失数据

正确做法：
✓ 联系IT申请正版授权
✓ 使用免费替代软件
✓ 使用开源替代软件
```
